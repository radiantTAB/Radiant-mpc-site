RADIANT QUIKRAM — DEMO NARRATION SCRIPT
Approximately 60 seconds. Paste into a voice-over tool, or record in your own voice.

Radiant QuikRAM is HDR and radioactive-materials management for radiation oncology — source inventory with automatic isotope decay correction, the full HDR Source Exchange QA worksheet, a well-chamber inventory with ADCL calibration values, a TG-43 line-source dose calculator, and printable date-AKS-activity decay tables, all in one local web app. Per source, QuikRAM records the manufacturer cal and decay-corrects the air-kerma strength to today using the isotope's published half-life. The HDR Source Exchange workflow walks you through every section of the quarterly QA — ODH 3701 acceptance checks, chamber constancy on the prior source, new source calibration, timer accuracy and reproducibility, timer linearity with an auto-computed R-squared plot, the source housing survey, and a source-positioning film upload — and rolls everything up into a summary that flags each test against editable tolerances. Print a date-by-date decay table for the source binder, or run a TG-43 dose calculation as an independent reference check. Radiant QuikRAM — HDR program management, precision physics in software.
