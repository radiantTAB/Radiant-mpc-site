Features

What QuikRAM does.

One place for HDR source records, byproduct material logs, and TG-43 reference calculations — with the audit trail surveys and NRC inspections actually ask for.

Source Inventory

Every sealed source on premises — HDR Ir-192 / Co-60, LDR Cs-137 historicals, I-125 / Pd-103 / Cs-131 seed lots, and Cs-137 check sources — with manufacturer, model, S/N, install date, and calibration certificate.

Automatic Decay Tracking

Per-source current air-kerma strength is decay-corrected from the calibration date using the published half-life for the isotope — today's activity is one click away, no spreadsheet.

HDR Source-Change Log

Every quarterly swap recorded: source out, source in, well-chamber verification, expected vs measured Sk, performed by, witness, and dispatch paperwork — a full audit trail per change.

TG-43 Dose Calculator

Line-source geometry, dose-rate constant Λ, radial dose function gL(r), and (perpendicular-axis) dose-rate output in cGy/h — using the parameters stored against each source model.

Printable Reports

Inventory snapshot, source-change audit trail, and per-source decay history — cleanly formatted for the source binder and NRC inspections.

Local-First

SQLite on the QA workstation — no cloud, no IT involvement. Same engine drives the standalone Windows app and the local web app, so the data stays where you work.

Who It's For

Made for HDR physicists & RSOs.

QuikRAM is built for the medical physicist or Radiation Safety Officer responsible for sealed-source possession — HDR Ir-192 sources cycling on a 13-week schedule, the historical LDR records that haven't been retired, the seed-implant lot inventory, and the check sources that survey meters depend on. It replaces the binder + spreadsheet combo with a structured record that answers the questions inspections actually ask.

The TG-43 dose calculator is a reference-grade second check — line-source geometry with stored per-model Λ and gL(r), decay-corrected from the source's calibration cert. Not a treatment-planning system; an independent hand-calc replacement that runs locally with no PHI.
