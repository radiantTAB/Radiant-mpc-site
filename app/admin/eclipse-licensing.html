Eclipse Script Licensing

Turn a clinic-neutral Eclipse/ESAPI script into a shippable, time-limited, site-locked bundle. Upload the script .zip, pick the clinic and term, and download the finished package — the license file is injected for you.

The bundle is assembled and downloaded locally. Issue a key for a product the first time and it's remembered for next time.
